# Planejamento de Implementação - Etapa por Etapa

Este documento serve para organizar e acompanhar o desenvolvimento das funcionalidades do PDV Nino com base nos tutoriais do GrandChef. Utilize os checklists abaixo para marcar seu progresso à medida que avança na implementação de cada módulo.

*   [x] **Ativando OAuth** – [Assistir ao vídeo](https://www.youtube.com/watch?v=CAnujKr-ifw&list=PLaWveaTRjHDdUSBSmKqzzvm2Jw8xaeguu&index=28)
